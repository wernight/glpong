Pong était le premier jeu sur ordinateur. Deux joueurs, chacun dirigeant son paddle, doivent se renvoyer la balle. Les points sont ici comptés comme au tennis. Le code source est fourni.

Le jeu se joue à 2 sur un même ordinateur. Le joueur gauche utilise les touches Shift et Ctrl, le droit utilise les flèches haut et bas.

Ce projet est issue des heures perdues en TP informatique. Après avoir fini mon TP, ne sachant que faire, j'ai voulus faire un tout petit jeu très simple. J'ai décidé de faire profiter la communoté de ce jeu, avec ses sources. Libre à vous de l'améliorer.

Fichiers:
- UML.vpp	Diagramme UML du projet visualisable avec Visual Paradigm.
- GLPong.sln	Solution pour Visual C++ 7
- Makefile	Makefile pour linux
- make.bat	Make simple pour Intel C++
- src/		Fichiers sources du jeu
- res/		Ressources (Windows)
- bin/		Fichiers compilés et finaux

Le jeu dépend de OpenGL, GLU et SDL.
